# Our Story — The Missing Flight

A mobile-first, cooperative 2.5D point-and-click puzzle adventure starring **Leah and Moshé**, spanning their whole journey: Israel, Europe, India, Thailand, Vietnam, Japan and South Korea — wrapped in one mystery about an erased boarding pass, a white rabbit, a brass Hamsa, and a flight that never left.

## The game

- **59 playable scenes** across a prologue, 7 chapters and a finale
- **63 story-integrated puzzles** (observation, logic, routes, timetables, wordplay, cooperative and dual-perspective), each with 4 progressive hints, reset and a solution explanation
- **Perspective switching** between Leah (engineer's eye) and Moshé (planner's eye), plus cooperative two-part and ordered dual-action puzzles
- **Mystery systems**: clue board, contaminated-memory anomalies in six categories, red herrings, chapter fragments and a hashed final answer — the solution is never stored in plaintext anywhere in this repository
- **DS-style save system**: 3 slots + autosave/backup recovery, rename/copy/delete (double confirmation), JSON export/import, versioned schema with migrations (including the old Chapter-1 prototype save)
- **Cutscene engine**: 10 data-driven PNG-layer cutscenes with skip/replay and a gallery (replays never touch the save)
- **Passport, Journey Seals, Rabbit Marks, route map, dialogue log**
- **PWA**: offline service worker, lazy chapter asset caching, installable
- **Accessibility**: captions, large text, high contrast, reduced motion, hotspot highlighting, keyboard focus styles, screen-reader labels, 44px touch targets

## Run locally

```bash
python3 -m http.server 8080   # then open http://localhost:8080
```

## Tests

```bash
node tests/run-tests.mjs
```

The suite validates the scene/chapter graph, puzzle validators, the clue dependency graph (no orphan clues, no missable mandatory clues), cutscenes, save migrations, every referenced PNG, and it **fails the build** on any SVG/PDF/WebP file or reference — or on any plaintext occurrence of the final answer.

## Art pipeline (placeholders)

All visuals are **PNG only** — SVG, PDF and WebP are prohibited and enforced by tests. Current art files are flat-colour placeholders generated by `tools/generate_placeholders.py`; the full art brief for every file (path, size, description) lives in `assets/ASSET_TEMPLATE.json`. Replace each placeholder with painted art at the same path.

- Leah: consistent face, tsniut — headscarf, long sleeves, long skirt in every sprite
- Moshé: curly dark hair, short beard, kippah visible in every sprite
