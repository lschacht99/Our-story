# Hamsa Nomads: The Missing Journey

A mobile-first point-and-click puzzle adventure based on Leah and Moshé’s travel story.

## Playable now

**Chapter 1 — Three Flights East**

Paris → Istanbul → Muscat → Mumbai

- Illustrated point-and-click scenes
- Leah and Moshé character art
- 10 interactive journey puzzles
- Progressive four-level hints
- Hamsa Nomads Passport
- route map and mystery notebook
- Rabbit Mark collectibles
- local autosave
- responsive mobile-first layout
- Havdalah rabbit memory ending

## Run locally

No build step is required.

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## GitHub Pages

Deploy the repository root from the `main` branch.
