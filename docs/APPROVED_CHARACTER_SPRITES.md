# Approved character sprites

The game now uses the user-approved Leah and Moshé artwork as transparent PNG sprite atlases.

## Atlas layout

Each atlas is a 4 × 2 grid:

1. Idle
2. Talking
3. Walking frame 1
4. Walking frame 2
5. Thinking / question
6. Surprised
7. Pointing / interacting
8. Jumping / celebrating

## Runtime files

- `assets/atlases/leah.png`
- `assets/atlases/moshe.png`
- `sprites.css`

The previous low-quality Leah and Moshé atlas files were removed and replaced. No SVG character artwork is used.
